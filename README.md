# vite-boilerplate

> A minimal project template for creating react apps with [Vite](https://vitejs.dev/)

![GitHub](https://img.shields.io/github/license/vahidtvj/vite-boilerplate)

## Features

- [TypeScript](https://www.typescriptlang.org/)
- [Prettier](https://prettier.io/) for formatting
- [ESLint](https://eslint.org/) for code linting
- [Stylelint](https://stylelint.io/) css linting
- [Storybook](https://storybook.js.org/) component development
- [husky](https://github.com/typicode/husky) + [lint-staged](https://github.com/okonet/lint-staged) managing commits

## Setup

- clone the repo or use `Use this template`
- run `yarn` to install necessary packages

## Development

- To run the app: `yarn dev`
- To develop components seperately: `yarn storybook`
